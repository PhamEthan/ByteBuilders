# REQUIREMENTS


# HOW TO BUILD PROGRAM 
1. npm install prisma @prisma/client pg bcryptjs express googleapis jsonwebtoken 
2. npx prisma init
3. npx prisma generate 

# DOCKER SET UP
1. docker compose build
2. docker compose run app npx prisma dev --name init 

# HOW TO RUN DOCKER/THE PROGRAM 
1. docker compose up - to turn off docker server
2. docker compose down - to turn off docker server

# HOW TO VIEW
1. Localhost:5003
